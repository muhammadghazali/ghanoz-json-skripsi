% menjelaskan obyek penelitian
\onehalfspacing CampusLife adalah \textit{mobile information directory application} yang dikembangkan oleh LayangLayang Mobile\footnote{http://layanglayangmobile.com} untuk menyediakan informasi yang relevan kepada civitas kampus. Untuk memenuhi kebutuhan penyediaan informasi kepada setiap penggunanya, maka LayangLayang Mobile perlu membangun Web API\footnote{http://en.wikipedia.org/wiki/Web\_API} untuk membuka akses data agar aplikasi CampusLife bisa menampilkan informasi relevan yang dibutuhkan oleh penggunanya.

% menjelaskan metode - metode yang ada
\onehalfspacing Semenjak tahun 2005, ProgrammbleWeb\footnote{Public API directory http://www.programmableweb.com} sudah melacak public web service yang berbasiskan REST dan SOAP dengan total persentase masing - masing untuk REST sebesar 69\% dan SOAP 22\%. Ini menunjukkan implementasi Web API berbasiskan REST jauh lebih populer dibandingkan SOAP.\cite{programmableweb-apis}

CampusLife akan didistribusikan kepada pengguna melalui salah satu \textit{mobile computing devices}, yaitu \textit{smartphone}. Mobile computing devices memiliki masa hidup baterai dan ketersediaan \textit{bandwidth} yang terbatas.\cite{challenging-issues-and-limitations-of-mobile-computing} Dengan kedua keterbatasan tersebut proses pertukaran data antara Web API dan CampusLife perlu dilakukan dengan efisien.

% menjelaskan rangkuman penelitian
\onehalfspacing Dalam penelitian ini penulis akan mengkaji penerapan format serialisasi data JSON untuk digunakan dalam proses pertukaran data antara Web API berbasiskan REST dan CampusLife mobile application. Hasil akhir yang diharapkan adalah format serialisasi data JSON yang digunakan mampu memberikan ukuran data yang lebih kecil dibandingkan format serialisasi data XML.
